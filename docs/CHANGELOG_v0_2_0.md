# Changelog — Build v0.2.0

## Novidades desta fase

- Tela inicial mantida com build, data e hora visiveis.
- Fluxo de novo jogo expandido:
  - escolha de avatar;
  - nome do gestor;
  - pais;
  - modo Realista ou Sandbox;
  - escolha obrigatoria de equipe de F2.
- Lobby principal expandido com abas:
  - Dashboard;
  - Pilotos;
  - Oficina;
  - Staff;
  - Instalacoes;
  - Calendario.
- Sistema inicial de patrocinadores.
- Sistema inicial de upgrades de carro.
- Sistema inicial de contratacao de staff.
- Calendario completo 2026 em estrutura editavel.
- Grid de pilotos baseado na lista enviada pelo usuario.
- Classificacao antes da corrida.
- Corrida 3D procedural em Miami com Three.js:
  - pista por waypoints;
  - carros 3D leves construidos em codigo;
  - cores por equipe;
  - movimento em tempo real;
  - pneus, combustivel, condicao e pit stop;
  - controles de ataque/poupar/pit stop.
- Resultado da corrida e classificacao do campeonato.
- LocalStorage para salvar carreira.
- Aviso de orientacao para forcar uso horizontal no celular.

## Observacao

O build nao inclui os assets pesados. Extraia os ZIPs de assets previamente baixados para formar a pasta `assets/` na raiz do projeto.
